Overview

    This extension indicates the EGL loader supports
    GLES layering on Android. When EGL starts up, the loader
    may call the following two functions, which must be
    exposed by layers:

    typedef __eglMustCastToProperFunctionPointerType(
        EGLAPIENTRY *PFNEGLGETNEXTLAYERPROCADDRESSPROC)(
        void *, const char *funcName);

    void AndroidGLESLayer_Initialize(
        void *layer_id,
        PFNEGLGETNEXTLAYERPROCADDRESSPROC next_gpa);

    void *AndroidGLESLayer_GetProcAddress(
        const char *funcName,
        __eglMustCastToProperFunctionPointerType next)

    The loader will only call those functions for
    selected applications, chosen by Android specific
    means.
